作业自动分类工具·使用说明

版本：V3.0

适用对象：GCC任课教师

————————————————————

一、工具简介

本工具用于根据学生成绩，自动整理“学习通下载的学生作业文件夹”。

1.自动创建成绩区间文件夹

2.自动按成绩移动学生作业

3.自动在文件夹名前标注成绩

二、适用条件

学生作业文件夹要求：

1.每个学生的作业为一个独立文件夹

2.命名格式必须为：学号-姓名

示例：202612340001-张三

说明：超星学习通批量下载并解压后的作业，默认即为该格式。

三、准备工作

打开“分文件夹.csv”，将学生的学号、姓名、成绩信息，按对应的列填入表格，并按原格式csv保存。

四、使用步骤（共4步）

第1步：运行程序

双击ScoreSorter.exe

第2步：选择学生作业文件夹

在弹窗中，选择 包含所有学生作业的文件夹，点击“确定”

第3步：选择成绩CSV文件

在弹窗中，在电脑中找到并选择“分文件夹.csv”，点击“确定”

第4步：等待完成

程序自动执行以下操作：

(1)创建成绩分类目录

(2)按分数移动学生文件夹

(3)在文件夹名前添加成绩

完成后会弹出提示框：处理完成。

六、常见问题

程序没有效果？

请检查：学生文件夹是否为“学号-姓名”；CSV中学号、姓名是否完全一致。

有学生未被分类？

程序会在同级目录生成“未匹配学生.txt”，可查看未匹配学生名单。

七、注意事项

1.程序运行时不要修改CSV

2.不要删除正在处理的文件夹

3.支持多次运行

4.不会破坏已分类结构
